viewVarAnnotation()

viewVarAnnotation(integer  $project_id, string  $patient_id, string  $sample_id = "null", string  $case_id = "any", string  $type) : \view

This function generates the view object for variant pages. The type of variants could be germline, somatic, rnaseq or variants (unpaired sample).

Use case I, showing variants at patient level:

Route => https://clinomics.ncifcrf.gov/production/public/viewVarAnnotation/22112/CL0036/null/20160415/germline

This URL will return germline mutation page of patient CL0035 of case 20160415.The mutation cohort will be the calculation of project 22112.

Use case II, showing variants at library level:

Route => https://clinomics.ncifcrf.gov/production/public/viewVarAnnotation/22112/CL0036/CL0036_N1D_E_HWMTYBGXX/20160415/germline.

This URL will return germline mutation page of library CL0035_T1D_E_HWJW5BGXX of case 20160415.

Use case III, showing merged variants

Route => https://clinomics.ncifcrf.gov/production/public/viewVarAnnotation/22112/CL0045/null/any/germline.

This URL will return merged germline mutation page for all cases.

Parameters

integer $project_id

project ID. We use this project ID to calculatr the gene/site cohort mutation percentage.
string $patient_id

patient ID
string $sample_id

sample ID. If view is at library level (exome or panel), this sample ID must be provided.
string $case_id

case ID. For merged view, use "any".
string $type

variant type: ['germline','somatic','rnaseq','variants']

Returns

\view —

pages/viewVarDetail view object

viewVarAnnotationByGene()

viewVarAnnotationByGene(integer  $project_id, string  $gene_id, string  $type, integer  $with_header, string  $tier_type = "null", string  $tier = "null", string  $diagnosis = "null", string  $patient_id = "null", string  $no_fp = "false", string  $maf = 1, string  $min_total_cov, string  $vaf) : \view

This function generates the view object for variant pages for specific gene. This view includes a lollipop plot. The type of variants could be germline, somatic, rnaseq or variants (unpaired sample).

Use case I, showing all MYCN germline variants in project 22112:

Route => https://clinomics.ncifcrf.gov/production/public/viewVarAnnotationByGene/22112/MYCN/germline.

This URL will return germline mutation page of gene MYCN.The mutation cohort will be the calculation of project 22112.

Use case II, showing variants for specific diagnosis:

Route => https://clinomics.ncifcrf.gov/production/public/viewVarAnnotationByGene/22112/MYCN/germline/0/null/null/Melanoma.

This URL will return germline mutation page of diagnosis Melanoma.

Use case III, showing variants with filters

Route => https://clinomics.ncifcrf.gov/production/public/viewVarAnnotationByGene/22112/BAP1/germline/0/germline/tier1/null/null/false/0.01/20/0.2.

This URL will return mutation page with filter {Tier: 'Tier1', MAF:0.01, : Totcal_cov: 20, VAF: 0.2}.

Parameters

integer $project_id

project ID. We use this project ID to calculatr the gene/site cohort mutation percentage.
string $gene_id

Gene symbol
string $type

variant type: ['germline','somatic','rnaseq','variants']
integer $with_header

if in iframe, set it 0 to avoid the menu bar in the iframe
string $tier_type

Tier type filter. Value could be 'germline' or 'somatic'
string $tier

Tier filer: ['Tier 1','Tier 2','Tier 3','Tier 4']. Default is the user setting
string $diagnosis

Diagnosis filer
string $patient_id

Patient filer
string $no_fp

Filer for false positive variants
string $maf

Filer for maximun population frequency
string $min_total_cov

Filer for total coverage
string $vaf

Filer for minimun VAF

Returns

\view —

pages/viewVarDetail view object
